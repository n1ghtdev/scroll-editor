## CSS WebKit Scrollbar generator.

### What can you tweak with this app:

- ::-webkit-scrollbar
  - width
- ::-webkit-scrollbar-track
  - width
  - background-color
- ::-webkit-scrollbar-thumb
  - width
  - background-color
  - border
  - border-radius
